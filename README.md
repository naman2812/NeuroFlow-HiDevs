# NeuroFlow-HiDevs

NeuroFlow is an architecturally complex system integrating document ingestion, vector-based retrieval, LLM generation, automated evaluation, and continuous fine-tuning.

## Directory Structure
- `backend/`: Core application logic and API endpoints.
- `frontend/`: Web interface for interacting with the system.
- `pipelines/`: Data ingestion, embedding, and processing pipelines.
- `evaluation/`: Automated LLM-as-a-judge evaluation frameworks.
- `infra/`: Infrastructure as code and deployment configurations.
- `docs/`: Architectural documentation, ADRs, and API contracts.

See `docs/architecture.md` for the system design.
