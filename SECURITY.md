# Security Policy

## Supported Versions
We currently support the latest major version of NeuroFlow. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within NeuroFlow, please do not disclose it publicly. Instead, please report it via the GitHub Security Advisory feature in this repository.

We take all security issues seriously and will respond to reports as quickly as possible, usually within 48 hours.

### What to include in your report
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and any suggested mitigations if known.
